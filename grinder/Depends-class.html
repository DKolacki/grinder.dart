An annotation to define a Task's dependencies.

Each listed dependency can be either a: link to a Function which is also annotated as a Task. Useful for rename refactoring, finding uses, etc. String. Useful for referring to programmatically added tasks. * TaskInvocation. Useful for passing args to dependencies.
