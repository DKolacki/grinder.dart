File getFile(String path)

Return the file pointed to by the given path. This method converts the given path to a platform dependent path.

Source 

File getFile(String path) {
  if (_sep == '/') {
    return new File(path);
  } else {
    return new File(path.replaceAll('/', _sep));
  }
}
